# 🎙️ Jarvis Voice Assistant

A Python-based voice assistant inspired by Siri and Jarvis that can recognize voice commands and perform web automation tasks. The assistant listens for the wake word **"Jarvis"**, responds to the user, and executes commands such as opening websites.

---

## ✨ Features

✅ Voice Recognition

✅ Text-to-Speech Responses

✅ Wake Word Detection ("Jarvis")

✅ Open Websites Using Voice Commands

✅ Hands-Free User Interaction

✅ Simple and Lightweight Implementation

---

## 🛠️ Tech Stack

- Python
- SpeechRecognition
- pyttsx3
- Webbrowser Module

---

## 🚀 Supported Commands

| Command | Action |
|----------|----------|
| Open Google | Opens Google |
| Open YouTube | Opens YouTube |
| Open LinkedIn | Opens LinkedIn |
| Open Instagram | Opens Instagram |

---

## ⚙️ How It Works

1. Start the application.
2. Say **"Jarvis"** to activate the assistant.
3. Give a command.
4. The assistant processes the command and performs the requested action.
